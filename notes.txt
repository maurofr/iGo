Els grafs tarden una barbaritat en descarregar-se, no oblidar guardar-los per no haver de descarregar-los cada cop.

Comandes git:

- git status -> et diu quines diferencies hi han entre els teus arxius i els de github
- git add . -> afegeix tots els canvis / fitxers nous per fer el commit
- git commit -m "missatge del commit" -> "contrasta" els teu add amb el github (no funciona si hi han conflictes si no recordo malament)
- git push -> s'actualitza el github amb els commits que tens fets
- git pull -> agafa els arxius / diferencies del github actualitzant la teva carpeta

hi ha el concepte de branches també pero crec que no ens farà falta, així les coses són més fàcils
